## Deskripsi tugas
Buatlah program menggunakan bahasa pemrograman PHP dengan
spesifikasi sebagai berikut:
â— Program bebas, kecuali program Ormawa
â— Menggunakan minimal 3 buah tabel
â— Terdapat proses Create, Read, Update, dan Delete data
â— Memiliki fungsi pencarian dan pengurutan data (kata kunci bebas)
â— Menggunakan template/skin form tambah data dan ubah data yang sama
â— 1 tabel pada database ditampilkan dalam bentuk bukan tabel, 2 tabel sisanya
ditampilkan dalam bentuk tabel (seperti contoh saat praktikum)
â— Menggunakan template/skin tabel yang sama untuk menampilkan tabel
